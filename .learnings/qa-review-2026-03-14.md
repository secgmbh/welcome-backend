#### 2. Memory Leaks in useEffect
**Dateien:** 
- `AdminPage.jsx` - setTimeout ohne Cleanup (showToast)
- `DashboardPage.jsx` - fetch ohne AbortController
- `LiveFeedPage.jsx` - mögliche Intervalle ohne Cleanup

**Risiko:** Memory Leaks wenn Komponenten unmounten während async Operationen laufen.

**Empfehlung:** 
```javascript
useEffect(() => {
  let isMounted = true;
  const controller = new AbortController();
  
  fetchData(controller.signal).then(data => {
    if (isMounted) setData(data);
  });
  
  return () => {
    isMounted = false;
    controller.abort();
  };
}, []);
```
